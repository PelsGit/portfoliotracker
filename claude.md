# Portfolio Tracker

An all-in one Portfolio performance management system for traders and investors. Track performance of investments and trades. 

## Stack
- **Backend:** Python 3.12 + FastAPI
- **Frontend:** React 18 + Recharts
- **Database:** PostgreSQL 16
- **Price data:** yfinance (Yahoo Finance)
- **Containerisation:** Docker Compose (local), Azure Container Apps / AKS (production)

See `/docs` for architecture and stack rationale.

## Public repository
This project will be hosted in a public github repository, ensure no secret values are included in the folders.
